==============================================================================
oli.areadme
==============================================================================

Simple Readme creator built with AngularJS

Features
--------

- Allows you to create simple Readme's


Installation
------------

Install oli.areadme by adding it to your buildout::

   [buildout]

    ...

    eggs =
        oli.areadme


and then running "bin/buildout"
